### 2. Setup Google Colab
1.  Open [Google Colab](https://colab.research.google.com/).
2.  Create a **New Notebook**.
3.  Rename the notebook (e.g., `my_analysis.ipynb`).
4.  **Organization:** In the file menu, locate the notebook in your Google Drive and move it to a specific folder dedicated to this course to keep things organized.

### 3. The Task (In the Notebook)
Write Python code to perform the following:
1.  **Load the data:** Read the data into a pandas DataFrame.
2.  **Save the file:** Save the raw data as a file (e.g., `.csv`) on your computer.
3.  **Process the data:** Perform basic data cleaning or pre-processing.
4.  **Visualize:** Create at least one graph/chart based on the data.

### 4. How to Submit (Save to GitHub)

**Part A: Upload the Data File**
1.  Go to this GitHub repository.
2.  Click **Add file** > **Upload files**.
3.  Upload the data file you saved in Step 3 (e.g., `data.csv`).
4.  Commit changes.

**Part B: Save the Notebook**
1.  Inside your Google Colab notebook, go to the top menu.
2.  Click **File** > **Save a copy in GitHub**.
3.  Authorize GitHub (if requested).
4.  Select **this repository** from the dropdown list.
5.  Click **OK**.
